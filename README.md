# Space Dungeon

A 2D Top-down Scrolling space shooter with a dungeon crawling element.

## Dependencies
- [SFML](https://github.com/SFML/SFML)
- [Doctest](https://github.com/onqtam/doctest)

## Building
Simply do:

```
make all
```
To build Spacedungeon and run it after building do:
```
make run
```
